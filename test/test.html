tailwindCSS 项目中使用 ​

1. 安装最新版tailwindCSS postcss autoprefixer ​

shell
pnpm install -D tailwindcss postcss autoprefixer
1

2. 使用向导初始化配置，生成 tailwind.config.js ​

shell
npx tailwindcss init
1

3. 添加postCSS配置 ​

js
module.exports = {
  plugins: {
    tailwindcss: {},
    autoprefixer: {},
  },
}

4. 修改 tailwind.config.js 配置 ​

js
import daisyui from 'daisyui'
/** @type {import('tailwindcss').Config} */
export default {
  // dark模式，不根据windos系统改变, 而是使用 html 节点的 class="dark"
  darkMode: 'selector',
  // 包含使用 taiwindCSS 的文件
  content: ['./index.html', './src/**/*.{vue,js,ts,jsx,tsx}', './docs/**/*.{vue,js,ts,jsx,tsx,md}'],
  theme: {
    extend: {},
  },
  plugins: [daisyui],
  daisyui: {
    prefix: 'da-',
  },
}

研究daisyui 主题 ​

在Layout.vue中添加如下代码，让isDark响应daisyui的主题

ts
const { isDark } = useData()
const isDaisyDark = useDark({
  selector: 'html',
  attribute: 'data-theme',
  valueDark: 'dark',
  valueLight: 'light',
  writeDefaults: isDark.value,
})

watch(isDark, () => {
  isDaisyDark.value = isDark.value
})
