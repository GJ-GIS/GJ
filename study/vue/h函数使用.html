关于Vue3使用h函数 ​

官网地址： https://cn.vuejs.org/guide/extras/render-function.html#rendering-slots

有两种方式使用--

第一种是使用渲染函数的方式；

第二种是使用对象配置的形式；

举一个例子，使用 h 函数实现一个 button 组件；

使用方式

ts
# App.vue
<script setup lang="ts">
import MyButton from "./MyButton.ts"
const onClick = () => {
  console.log('onClick')
}
</script>
<template>
  <MyButton :disabled="false" @custom-click="onClick">
    my button
  </MyButton>
</template>

渲染函数的实现 ​

渲染函数也可以在当前（App.vue）组件中定义使用。

ts
import { defineComponent, h } from 'vue'
function MyButton(props, { emit, slots }) {
  return h(
    'button',
    {
      disabled: props.disabled,
      onClick: evt => emit('custom-click', evt),
    },
    slots.default(),
  )
}
MyButton.props = ['disabled']
MyButton.emits = ['custom-click']
export default MyButton

配置项的实现 ​

可以不用 defineComponent ; 这个 api 主要是实现 ts 的类型推导的，返回传递的对象，一般在ts文件中使用较多。

ts
# MyButton.ts
import {defineComponent, h } from 'vue'
export default defineComponent ({
  name: 'MyButton',
  props:['disabled'],
  emits: ['custom-click'],
  setup(props, context) {
    return () => h(
      'button',
      {
        onclick:(e:Event) => {context.emit('custom-click',e)},
        disabled: props.disabled,
      },
      context.slots.default())
  }
})

写法一： 渲染函数 ​

ts
import { FunctionalComponent, h } from 'vue'
export interface TreeData {
  key: string
  title: string
  children: TreeData[]
}
const TreeComponent: FunctionalComponent<{ data: TreeData[] }> = props => {
  const { data } = props
  if (!data || !data.length) return
  const item = data.map(v => {
    const { key, children, title } = v
    const node = [h('label', null, title)]
    if (children && children.length) {
      node.push(TreeComponent({ data: children }))
    }
    return h('li', { key }, node)
  })
  return h('ul', null, item)
}

export default TreeComponent

写法二： 模板组件 ​

vue
// TreeComponent.vue
<script setup lang="ts">
  interface TreeData {
    key: string
    title: string
    children: TreeData[]
  }
  defineProps<{ data: TreeData[] }>()
</script>

<template>
  <ul>
    <li v-for="el in data" :key="el.key">
      {{ el.title }}
      <TreeComponent v-if="el.children?.length" :data="el.children"></TreeComponent>
    </li>
  </ul>
</template>
